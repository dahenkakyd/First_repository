Автоматизация тестирования

Необходимо выполнить команду: pytest --alluredir results (создание папки с результатами тестов)

Затем выполнить команду: allure generate final-report (отчет с файлами тестирования)

Для автоматического запуска тестов выполнить команду в консоли ./run.sh Отчет автоматически откроется в вашем браузере
